/*
引入字体图标步骤：
1、登录icomoon字库 https://icomoon.io/ 选择图标后下载，得到icomoon.zip压缩包；
2、解压后，复制fonts目录到项目工程目录；
3、打开解压目录下的style.css样式文件，将 @font-face { ... ... } 内容整体复制到html的style中，引入字体；
4、打开解压目录下的demo.html网页，复制需要的图标后面的 “” 方块图标，填入需要的html标签中；
5、为目标标签指定字体font-family: 'icomoon';即可
*/

/*
追加新字体图标方法：
1、打开https://icomoon.io/，在选择图标页面左上角有一个导入按钮 “Import Icons”；
2、弹出对话框中选择之前解压目录下的selection.json文件；之后会发现之前的图标会以增量的方式加入到选择区；
3、之后可以继续挑选需要的其他图标，选好后下载并使用，和之前的方式一样；
*/
